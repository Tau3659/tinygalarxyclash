# 游戏音频文件说明

## 文件要求

请将你准备好的音频文件放在这个文件夹中：

### 背景音乐
- **文件名**: `background.mp3`
- **用途**: 游戏背景音乐，循环播放
- **建议**: 轻松的科幻或太空主题音乐
- **时长**: 建议1-3分钟，支持无缝循环

### 射击音效
- **文件名**: `shoot.mp3`
- **用途**: 无人机发射子弹时的音效
- **建议**: 短促的激光或能量武器音效
- **时长**: 建议0.1-0.5秒

## 音频格式

游戏使用 **MP3** 格式，具有：
- 广泛的浏览器支持
- 较小的文件大小
- 稳定的播放性能

## 文件大小建议

- **背景音乐**: 建议不超过5MB
- **音效**: 建议不超过100KB

## 使用说明

1. 将音频文件复制到此文件夹
2. 确保文件名正确（background.mp3 和 shoot.mp3）
3. 重新启动游戏服务器
4. 在游戏中通过设置按钮调整音量

## 注意事项

- 现代浏览器需要用户交互才能播放音频
- 背景音乐会有3秒的音量渐变效果
- 可以在设置中单独控制背景音乐和音效的开关及音量
