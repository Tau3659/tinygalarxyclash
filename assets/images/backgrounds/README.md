# 背景图片说明

## 图片放置

### 欢迎界面背景
请将太空背景图片重命名为 `space_background.jpg` 并放置在此目录中。

### 游戏背景
请将星空背景图片重命名为 `game_background.jpg` 并放置在此目录中。

## 图片要求
- **欢迎界面**: `space_background.jpg` - 用于游戏启动界面
- **游戏背景**: `game_background.jpg` - 用于游戏进行时的背景
- 格式：JPG 或 PNG
- 建议分辨率：1920x1080 或更高
- 图片会自动缩放以充满整个窗口，保持比例不变

## 当前效果
- ✨ 欢迎界面：太空背景 + 100个闪烁星星 + 毛玻璃UI
- 🎮 游戏界面：星空背景 + 200个装饰星星 + 游戏元素
- 🌌 背景图片自动适配屏幕尺寸，保持比例

## 文件结构
```
assets/images/backgrounds/
├── space_background.jpg   # 欢迎界面背景（需要放置）
├── game_background.jpg    # 游戏背景（需要放置）
└── README.md             # 此说明文件
```

## 备用方案
如果背景图片加载失败，系统会自动使用默认的深色背景。
